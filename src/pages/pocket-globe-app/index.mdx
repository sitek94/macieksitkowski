# Pocket Globe App

<ExternalLinks
  liveUrl="https://sitek94.github.io/pocket-globe-app"
  githubUrl="https://github.com/sitek94/pocket-globe-app"
/>

In this app you can use highly interactive and responsive globe to select a
country. Then the app will connect with various APIs to get more details about
the country you've selected.

## Technologies used

- React - interactive UI
- D3 - dynamic and interactive data visualizations
- World Atlas TopoJSON - redistribution of
  [Natural Earth](http://www.naturalearthdata.com/)’s vector data
- Material UI - a collection of React components
- REST Countries - information about countries via RESTful API
- Unsplash API - free high-resolution photos
- Wikimedia REST API - access to Wikimedia's content and metadata

## Keyboard shortcuts

| Key      | Description                |
| -------- | :------------------------- |
| `ctrl ↑` | Rotate up                  |
| `ctrl ↓` | Rotate down                |
| `ctrl ←` | Rotate left                |
| `ctrl →` | Rotate right               |
| `+`      | Zoom in                    |
| `-`      | Zoom out                   |
| `l`      | Rotate to selected country |
| `r`      | Select random country      |
| `w`      | Show/hide widgets          |
| `ctrl /` | Show/hide shortcuts        |
